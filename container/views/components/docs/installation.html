Installation

After forking or cloning the repository,

  • simply setup your .env file, or set environment variables on your system
  • run installation, to install dependencies
> npm install

Starting the server

you can run the server file, or you can declare your server
start script inside package.json file on [scripts.start] then run as

npm start

OR

 // run 
 >>   node server.js
 //or if on development, you can install nodemon then run
 >>   nodemon server.js
